## Customization

### Changing Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #6366f1;      /* Main brand color */
    --secondary-color: #8b5cf6;    /* Secondary color */
    --text-color: #1f2937;         /* Main text color */
    --text-light: #6b7280;         /* Light text color */
    --bg-light: #f9fafb;           /* Light background */
}
```

### Adding Content

- Edit `index.html` to change text, sections, and structure
- Modify `styles.css` to adjust styling
- Update `script.js` to add new functionality

### Contact Form

The contact form currently shows an alert. To make it functional:

1. Set up a backend service (like Formspree, Netlify Forms, or your own server)
2. Update the form submission handler in `script.js`
